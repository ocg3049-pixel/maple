# 외부 리소스 보관소

MSW 기본 리소스에 없어서 **외부에서 직접 구해온 이미지/사운드 원본**을 여기에 보관한다.
(MSW에 업로드하고 나면 게임은 RUID만 참조하지만, 원본이 없으면 나중에 다시 못 만든다.)

## 폴더 규칙

```
resources/
  <스킬명 또는 기능명>/
    원본 파일들 (1.png, hit1.png, icon.png …)
    RUID.md            ← 파일 ↔ 업로드된 RUID 대응표 (필수)
```

## 새 외부 리소스를 가져왔을 때

1. `resources/<이름>/` 폴더를 만들고 원본 파일을 그대로 복사한다.
2. `mcp__msw-mcp__asset_create_account_resource_storage_item` 2단계 업로드로 RUID를 받는다.
   (presigned URL 요청 → 파일 PUT → fileUrl 붙여서 재호출)
3. 받은 RUID를 `RUID.md`에 **파일명 기준으로** 기록한다. 코드에 RUID를 박아넣기 전에 여기부터 적는다.

## 목록

| 폴더 | 내용 | RUID 표 |
|---|---|---|
| `새비지블로우/` | 새비지블로우(2차 스킬) 모션 8프레임 + 히트 이펙트 3종 + 아이콘 | [RUID.md](새비지블로우/RUID.md) |
| `돌진/` | 돌진(4차) 노란 빔 5프레임 + 흰 섬광 6프레임 + 아이콘 (구버전 클라 WZ 추출) | [RUID.md](돌진/RUID.md) |

> MSW 라이브러리에는 **구버전 아트가 없다**(현대 클라 덤프 기반). 사운드는 구버전 그대로 남아있는 경우가
> 많으니 업로드 전에 `../msw-library-index/`를 먼저 확인할 것.

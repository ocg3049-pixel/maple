# 브랜디쉬 강화(4차 버프) - 업로드 RUID 대응표

브랜디쉬 강화 스킬: 5분 지속버프(부스터식). 활성 중 브랜디쉬 데미지 1.5배 + 공속 +1단계 +
시전 이펙트 파워스트라이크 궤적색(선명한 살몬-주황). SkillManager 4차 목록 등록,
CharInfoManager.UseBrandishEnhance / IsBrandishEnhanceActive.

## 아이콘

브랜디쉬 아이콘(라이브러리 `185d0cd674bd44f388c8ccd265b59f43`)의 썸네일을 받아
**파워스트라이크 궤적색(주황 hue~23°, 채도~0.84)**에 맞춰 재색상. 처음엔 듀오톤(단색)으로 만들었다가
"너무 단색이라 부자연스럽다 + 파워스트라이크 궤적색과 다르다"는 피드백으로, **색상(hue)만 주황 밴드로
회전하고 명도/상대채도를 보존**하는 방식으로 교체 → 검·음영·에너지의 색 다양성이 살아있는 자연스러운
주황 아이콘. (동일 RUID로 이미지만 update)

> ⚠ 동일 RUID로 이미지만 교체(update)하면 클라 캐시가 갱신 안 됨(핑크→주황 반영 안 돼 실측 확인). 색 바꿀 땐
> **새 RUID로 업로드**하고 카탈로그 ruid를 교체할 것. 이미 퀵슬롯에 배정돼 있던 스킬은 저장된 옛 RUID를 물고
> 있어 아이콘이 옛 색으로 남으므로, 스킬창에서 슬롯으로 다시 드래그하면 새 아이콘으로 갱신된다.

## 이펙트 색

시전 이펙트(브랜디쉬 주황 아크 `1c29e2f0…`)는 별도 업로드 없이 런타임 Color 틴트로 진한 빨강 표시(요청):
PlayBrandishCastEffect에서 enhanced일 때 `Color(1.0, 0.13, 0.13, 1.0)` 곱셈 적용.

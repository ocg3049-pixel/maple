# 스피어버스터 강화 (4차) - 전용 이펙트 아트 RUID

사용자가 새로 그린 흑백 전용 아트. 기존 "어드밴스드 스피어 버스터"의 파랑→검정 재색본
(msw-dev-tools/resources/어드밴스드스피어버스터/)을 **완전히 대체**한다.
각 이미지는 원본 스피어버스터 클립 프레임과 **정확히 동일한 픽셀 크기**로 그려져 슬롯에 1:1 대응된다.
모두 asset_create_account로 업로드(sprite/skill) → **중심피벗**.

## 적용 위치 (PlayerAttack.mlua)

- `AttackSpearBurst`: `wingRuid`(날개), split-growth `circleRuid`/`headRuid`(마법진→용머리).
  ⚠ 새 아트는 중심피벗이라 split-growth `pivotOffset`을 enhanced일 때 0으로 분기(원본 파랑 클립은 좌하단 피벗 보정 유지).
- `SpawnAdvSpearStabEffect`: 찌르기 3종 용머리(variant 1/2/3).
- `SpawnAdvSpearHitFrames`: 히트 이펙트 3프레임(009→004→010)을 0.07초 간격으로 갈아끼워 재생.
  강화가 아닐 때만 원본 클립(84c06a06)을 `PlaySpearBurstHitEffect`로 재생한다(검붉은 틴트는 폐기).
  asset 업로드는 sprite만 가능해 새 애니메이션클립을 못 만들므로 프레임 교체 방식을 쓴다.
- 활성화 게이트: `CharInfoManager.advancedSpearBusterOwned = true`.
